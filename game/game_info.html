About:

This is a game of luck, the closer you get to your desiered number the more likely you are to win! Change the values to make it easier or harder.
This game was made for CSC102 at UAT

How to play:

1. Select your Die type
* d4
* d6
* d8
* d20
2. Chose game type
* Average
* Sum
* Max
* Min
3. Chose goal
* This is a number that you are wanting to get
(Number: 0-∞)
4. Grace
⚠ If unsure leave at default ⚠
* This value allows you to make it more or less difficult, the higher the number the more likely you are to win
(Number: 0-100)
